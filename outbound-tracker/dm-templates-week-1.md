## What We're Testing

1. **Does the distrust-first angle resonate with AI tool founders?** (They should recognize the problem)
2. **Does the portfolio case studies feel credible?** (Do they want to book based on one artifact?)
3. **What questions do they ask back?** (Signals what's actually hard in their domain)
4. **Reply rate:** If <5%, messaging isn't hitting. If >15%, we're onto something.

## Adjustments for Week 2 (Based on Week 1 Feedback)

- If founders ask "What's your process?" → emphasize speed (can start in 2 weeks)
- If founders focus on "trust with non-technical stakeholders" → pivot messaging to emphasize that angle
- If founders reply with skepticism ("We don't have budget") → ask if they're hiring, adjust positioning
- If reply rate >10% → expand list, add more Tier 1 founders before moving to Tier 2
- If reply rate <5% → rewrite templates, test different angles (speed vs. credibility vs. process depth)
