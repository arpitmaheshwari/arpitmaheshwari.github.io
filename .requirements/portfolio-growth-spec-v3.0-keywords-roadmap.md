### Week 2-3: Case Study Optimization (Parallel)
**Goal:** Add searchable text to 3-4 flagship case studies  
**Owner:** You (with design/UX support if needed)

**Tasks:**
- [ ] Select 3-4 strongest case studies (best visual portfolio pieces)
  - [ ] Map each to a secondary keyword:
    - Case Study A → "Data-intensive product design"
    - Case Study B → "Machine learning interface design"
    - Case Study C → "Fintech UX designer" (if applicable)

- [ ] For each case study, create text content:
  - [ ] **Problem Statement (150-200 words):** What was the challenge? Why did it matter?
    - *Example:* "Built the design system for an AdTech platform helping advertisers interpret ML model confidence scores in their optimization workflow. The core challenge: making black-box ML decisions transparent to non-technical users."
  - [ ] **Design Approach (250-300 words):** How did you approach it? What methodology? Key decisions?
    - *Example:* "Started with user research to understand mental models... Designed a progressive disclosure system... Iterated on visual language..."
  - [ ] **What Changed (100-150 words):** What shipped? How did it impact the product/experience?
    - *Example:* "Introduced explainability layer that helped users understand model decisions without requiring technical expertise. Simplified decision trees from 7 layers to 3."
  - [ ] **Industry Context:** What sector? What's unique about that domain?
    - *Example:* "AdTech platforms struggle with model transparency at scale..."
  
  - [ ] **Credibility Signal:** Add 1 of these (if available):
    - Testimonial quote (even generic: "Transformed how we approach design")
    - Company name + role context ("Lead Designer at [Sector] platform")
    - Years in sector ("8 years designing for AdTech")
    - Awards/recognition (if applicable)

**Deliverable:**
- 3-4 case studies with 600-800 words searchable text each
- Ready to be published as individual landing pages
